Star ratings of Los Angeles restaurants

Review and rating data courtesy of the Yelp developer api. The Python and Plotly scripts for the api pull and plotly map (clickable below) are available in this GitHub repository: LA_Restaurants_Yelp_API . Data cleanup and visualization in Python Pandas notebooks are here: stars_of_la GitHub repository

20,000 restaurants randomly selected.

Yelp's api limits information to 1,000 data points at a time. Since LA is home to over 300 zipcodes, by using only those that start with 900, information was collected from the communities closest to city center.
